Description:
A Notepad app that was created as part of a module learnng about andorid programming, the app uses a listview to display any movie details that the users include, a toggle button that sorts by Age rating and also automatically assigning the icons for the rating. Although looking plain and simple, I really did enjoy creating this application and would love to learn more about android programming, in the future.
