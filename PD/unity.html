Escape room


Year created:
2024

Languages/Tools used:
C# scripting, Unity

Description:
This ongoing project is an immersive Virtual Reality (VR) puzzle game developed using Unity, showcasing my skills in game design, VR development, and C# scripting. The primary focus of the game is a bookshelf puzzle, offering players an engaging challenge where the correct combination of book pushes unlocks a door. Using the power of ProBuilder for constructing both the book designs and the room environment.The level design process was made more efficient using ProBuilder, which made it possible to create and modify 3D objects directly in the Unity editor. Furthermore, well chosen imported assets improve the overall visual and add to the immersive atmosphere of the game.
