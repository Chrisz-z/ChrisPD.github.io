Movie4U

Year created:
2023

Languages/Tools used:

Description:
Movie4U is a basic movie website created using Bootstrap, CSS, HTML, and SQL. The website primarily focuses on providing information about movies, including their titles, genres, release years, synopsis and reviews. All information display are taken from a database an updated real-time While the design may not be aesthetically advanced, it serves as a functional platform for viewing, login, registering, viewing/adding/deleting reviews.
